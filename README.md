# Status Now — Redesigned UI

Hebrew-first, RTL, responsive Vite/React application. The project retains the existing Netlify and Vercel deployment configuration.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy

Push the project to the connected GitHub repository. Netlify or Vercel will build it using the existing project configuration.

## Main redesign changes

- Stronger hierarchy and clearer live-status cards
- Responsive desktop sidebar and mobile bottom navigation
- Search and category filtering
- Interactive likes and working report composer
- Live map preview and community-trust messaging
- Centralized styling in `styles.css`
